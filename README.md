ロリコンのためのプログラミング言語、Loli God
====

ロリロリ神、降臨

仕様
----

Brainfuck系言語です

- ういちゃん！ういちゃん！ - ポインタを右に移動
- 待ってよういちゃん！ - ポインタを左に移動
- かわいいよォ～！ - ポインタの指す値を1増やす
- ういちゃん... - ポインタの指す値を1減らす
- うい、こっち向いてェ～～～ - ポインタの指す値を出力する
- ごめんなさーい - 入力から1バイト読み込んで、ポインタが指す値に代入する
- 助かる～ゥ - ポインタの指す値が0なら、後の 'あぁぁあぁぁぁ～～～' までジャンプする
- あぁぁあぁぁぁ～～～ - ポインタの指す値が0でなければ、前の '助かる～ゥ' までジャンプする


それ以外の文字や改行は全て無視されます
